Class 3Z Ginpachi-sensei

Class 3Z Ginpachi-sensei is a light novel written by Ohsaki Tomohito based on Gintama created by Sorachi Hideaki. Class 3Z Ginpachi-sensei is a parody version of Class 3B Kinpachi-sensei. In Gintama version, Sakata Gintoki is a language teacher in Gintama High School. Most of the cast in Gintama, including the Yorozuya, Shinsengumi,.. etc are students in Gintama High School. In addition, the anime has animated some illustrations for Class 3Z Ginpachi-sensei as a side bonus at the end of some episodes so that Ginpachi-sensei (Sakata Gintoki) would answer any questions asked by readers.
